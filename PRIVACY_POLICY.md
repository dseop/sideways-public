# 개인정보 처리방침 (Privacy Policy)

**마지막 업데이트:** 2026년 5월 1일

'Sideways'(이하 '앱')는 사용자의 개인정보를 소중하게 생각하며, 관련 법령을 준수합니다.

## 1. 수집하는 개인정보 항목
본 앱은 사용자의 어떠한 개인식별정보도 서버로 전송하거나 수집하지 않습니다. 앱 운영에 필요한 다음의 권한만을 사용합니다:
- **카메라 (Camera):** 사진 촬영 기능을 제공하기 위해 필수적으로 접근합니다.
- **사진 라이브러리 (Photo Library):** 촬영한 사진을 기기에 저장하고 앱 내 갤러리에서 보여주기 위해 접근합니다.

## 2. 개인정보의 활용
- 앱에서 처리되는 모든 이미지 데이터는 **사용자의 기기 내에서만 처리**됩니다.
- 외부 서버로 이미지를 업로드하거나 공유하지 않습니다.

## 3. 제3자 제공 및 위탁
본 앱은 사용자의 데이터를 제3자에게 제공하거나 외부 기관에 처리를 위탁하지 않습니다.

## 4. 데이터 보안
모든 사진 데이터는 iOS의 표준 보안 체계 내에서 안전하게 관리됩니다.
